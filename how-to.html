단위 변환기 사용 방법

몇 번의 클릭만으로 모든 단위를 변환하는 방법을 알려드립니다.

1단계: 변환할 단위 유형 선택

메인 화면 상단에 있는 6개의 카테고리 버튼 중에서 변환하려는 단위 유형을 선택하세요.

  • 길이 - mm, cm, m, km, in, ft, yd, mile
  • 무게 - g, kg, ton, lb, oz
  • 온도 - 섭씨 (°C), 화씨 (°F), 켈빈 (K)
  • 속도 - km/h, mph, knot, mach
  • 면적 - m², ft², acre, ha
  • 부피 - mL, L, gal, cup

검색창에 "길이", "온도" 등을 입력하면 원하는 카테고리를 빠르게 찾을 수 있습니다.

2단계: 값과 단위 입력

선택한 카테고리의 변환기 카드에서 다음을 설정하세요:

  • 변환할 값 - 숫자를 입력합니다. 소수점도 입력 가능합니다.
  • 출발 단위 - 드롭다운에서 변환하려는 단위를 선택합니다.
  • 도착 단위 - 드롭다운에서 결과를 표시할 단위를 선택합니다.

값을 입력하는 순간 실시간으로 변환 결과가 표시됩니다.

3단계: 실시간 결과 확인

값을 입력하면 즉시 두 가지 방식으로 결과가 표시됩니다:

  • 주요 결과 - 선택한 도착 단위로 변환된 값이 결과 입력창에 표시됩니다.
  • 전체 변환값 - 카드 하단에 모든 단위로의 변환 결과가 한눈에 표시됩니다.

팁: 출발 단위와 도착 단위를 자유롭게 변경하면서 다양한 변환을 시도해보세요. 모든 변경사항은 실시간으로 업데이트됩니다.

활용 팁

  • 해외 여행 - 현지 속도 제한(mph)을 km/h로 변환하거나, 온도(°F)를 °C로 변환하세요.
  • 요리 - 해외 레시피의 컵, 온스 단위를 mL, g으로 변환하세요.
  • DIY/인테리어 - 인치와 피트 단위를 cm, m로 변환하여 정확하게 측정하세요.
  • 운동 - 파운드(lb)를 kg으로 변환하여 체중이나 운동 중량을 확인하세요.
  • 부동산 - 제곱피트(ft²)를 제곱미터(m²)로 변환하여 면적을 비교하세요.
